# Bus Ticket Booking Platform

A modern, blockchain-powered bus ticket booking platform that offers secure, transparent, and efficient ticket booking services. Built with React, Ethereum, and modern web technologies.

## 🌟 Features

- **Blockchain Integration**
  - Ethereum-based ticket minting
  - Secure payment processing
  - Transparent transaction history
  - NFT-based ticket ownership

- **Multiple Payment Options**
  - Ethereum (cryptocurrency)
  - UPI (PhonePe, Google Pay, Paytm)
  - Credit/Debit Cards

- **User Experience**
  - Modern, responsive design
  - Dark/Light mode support
  - Real-time seat selection
  - Interactive bus layout
  - AI-powered chatbot support

- **Security Features**
  - Smart contract-based ticket validation
  - Encrypted payment processing
  - Secure user authentication
  - Fraud prevention mechanisms

## 🚀 Tech Stack

- **Frontend**
  - React.js
  - Tailwind CSS
  - Framer Motion
  - React Router
  - React Icons

- **Blockchain**
  - Ethereum
  - ethers.js
  - MetaMask Integration
  - Smart Contracts

- **Backend Services**
  - Firebase Authentication
  - UPI Payment Gateway
  - Ticket Management System

## 📦 Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/bus-ticket-hackathon.git
cd bus-ticket-hackathon
```

2. Install dependencies:
```bash
npm install
```

3. Create a `.env` file in the root directory:
```env
REACT_APP_CONTRACT_ADDRESS=your_contract_address
REACT_APP_NETWORK_ID=11155111 # Sepolia Testnet
```

4. Start the development server:
```bash
npm start
```

## 🔧 Configuration

### Smart Contract
- Deploy the smart contract to Sepolia Testnet
- Update the contract address in `.env`
- Ensure MetaMask is configured for Sepolia Testnet

### Payment Gateways
- Configure UPI payment gateway credentials
- Set up card payment processing
- Configure Ethereum wallet integration

## 🎯 Usage

1. **User Registration**
   - Sign up with email/password
   - Connect MetaMask wallet
   - Complete profile setup

2. **Booking Process**
   - Select departure and destination
   - Choose travel date
   - Select bus and seats
   - Choose payment method
   - Complete transaction

3. **Ticket Management**
   - View booked tickets
   - Download ticket PDF
   - Track bus location
   - Cancel bookings (if applicable)
